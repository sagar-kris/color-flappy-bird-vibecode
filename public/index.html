Welcome to Flappy Bird

Instructions:

  • Press the "up" arrow key to make the bird fly up.
  • Press the "down" arrow key to make the bird fly down.
  • Avoid the pipes and stay within the screen boundaries.
  • The "game mode" toggle at the top left dictates if the current mode of the game is "text mode" or "color mode". This toggle changes randomly every 5 seconds.
  • Words pop up every 5 seconds on the screen. Before the refresh timer hits the first black line (3 seconds), you have to say out loud the word text if the game is in "text mode", or say out loud the word color if the game is in "color mode".
  • Basically, speak the correct color or text of the word that pops up on the screen based on the game mode. You have 3 seconds to do this. (Mode = ___ at the top left)
  • The options for word text / colors are 'red', 'green', 'blue', 'yellow', 'purple', 'orange', 'pink', 'brown', 'black', 'white'.
  • If you are correct, congrats. If you are not... the controls are locked for the last 1 second of the timer until the timer refreshes. Your bird is frozen!
  • Have fun!
  • Note, the first time the game starts, the browser will ask you for microphone permission. Click allow for this website. Browser-wise, Google Chrome works best.
